## Setup

### Pre-request

Debian based:

```
sudo apt install -y cmake flex bison
```

Arch based:

```
sudo pacman -S cmake flex bison
```

### Install

Using cmake

```
cmake -S . -B .
```

## Run

Compile

```
make
```

Run

```
./PCV
```
